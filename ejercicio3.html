border-radius: Esta propiedad establece el radio de los bordes de un elemento, controlando la curvatura de las esquinas.

border: La propiedad border establece el ancho, estilo y color de los bordes de un elemento en un solo atajo.

dotted: Esta propiedad establece un borde punteado para un elemento.

dashed: Esta propiedad establece un borde discontinuo para un elemento.

solid: Esta propiedad establece un borde sólido para un elemento.

double: Esta propiedad establece un borde doble para un elemento, que consiste en dos líneas paralelas separadas por un espacio.

groove: Esta propiedad establece un borde en relieve para un elemento.

ridge: Esta propiedad establece un borde en relieve inverso para un elemento.

inset: Esta propiedad establece un borde con sombra interior para un elemento.

outset: Esta propiedad establece un borde con sombra exterior para un elemento.

none: Esta propiedad elimina completamente el borde de un elemento.

hidden: Esta propiedad oculta el borde de un elemento, manteniendo el espacio que ocuparía el borde si estuviera presente.
