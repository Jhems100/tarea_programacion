RGB = Red 255, 0, 0

RGB = Green 0, 128, 0

HEX = Blue: #0000ff

HEX = Yellow: #FFFF47

RGB: este uso de colores para el fondo de la página web, utiliza la mezcla de los colores (RED, GREEN Y BLUE) en una escala de 0 - 255 cada uno. Ejemplo: (255, 0, 0) es una mezcla de (RED, GREEN, BLUE) y representa el color rojo

HEX: este uso de colores para el fondo de la página web, utiliza el sistema hexadecimal que es una representación de color que utiliza una combinación de seis caracteres alfanuméricos que van desde 0 a 9 y A a F. Ejemplo: (#0000ff) es el código que representa el color azul
